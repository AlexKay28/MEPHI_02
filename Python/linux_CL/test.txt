Какая команда без ключей (опций) позволяет увидеть список директорий, в которых последовательно ищется соответствующий файл для запуска?
echo $PATH

Какая команда с указанием только одного ключа (опции) позволяет получить список всех активных процессов системы?
ps -[eA]

Конвейер каких команд, имеющий минимальную длину, позволяет узнать общее количество строк во всех 
файлах с расширением py в домашней директории и всех её поддиректориях?
find ~ -name \*.py -exec cat {} + | wc -l

Какая команда минимальной длины позволяет перейти в родительскую (по отношению к текущей) директорию?
cd ..

Какая команда минимальной длины позволяет постранично просматривать содержимое файла file.txt в текущей директории?
(less|more) file.txt

Конвейер каких команд, имеющий минимальную длину, позволяет узнать количество файлов с расширением py в домашней директории и всех её поддиректориях?
find ~ -name *.py |wc -l

Какая команда без ключей (опций) позволяет увидеть ранее выполненные команды?
history

Какая команда минимальной длины позволяет (при наличии соответствующих прав) получить количество байт в файле file.txt, находящемся в текущей директории?
wc -c file.txt

Какая команда минимальной длины позволяет (при наличии соответствующих прав) удалить директорию test в текущей директории вместе 
со всем ее содержимым?
rm -r test

В каждой строке файла file текущей директории содержится одно число. Какая команда минимальной длины позволяет отсортировать
содержимое этого файла в порядке возрастания чисел?
sort -n file 

Какая команда минимальной длины позволяет дописать в существующий в домашней директории файл text.txt текст, вводимый с клавиатуры?
cat >> ~/text.txt

В каждой строке файла file домашней директории содержится одно слово. Конвейер каких команд, имеющий минимальную длину,
позволяет получить количество различных слов в этом файле?
sort ~/file | uniq | wc -l

Какая команда минимальной длины, не использующая цифр, позволяет сделать текущую директорию «проходимой» и доступной для
чтения всем пользователям системы в предположении, что все директории по пути к корню файловой системы таковыми уже являются?
chmod +rx ./

Какая подстановка результата выполнения одной команды в другую, имеющая минимальную длину, позволяет получить общее количество строк во всех файлах с расширением py в домашней директории и всех её поддиректориях?
wc -l 'find ~ -name \*.py'

Какая команда (при наличии соответствующих прав) безоговорочно прекращает выполнение процесса с идентификатором 29350?
kill (9|-s SIGKILL| --signal SIGKILL) +29350 

Какая команда минимальной длины позволяет получить список всех обычных файлов (regular files) в текущей директории и её поддиректориях, изменённых не более 30 минут назад?
find -type f -mmin -30

Какая команда без ключей (опций) позволяет получить справочную информацию (документацию) по команде command?
man command
